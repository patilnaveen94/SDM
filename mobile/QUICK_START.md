# 🚀 Quick Start - SDM Mobile App

## Run the App (3 Steps)

### 1️⃣ Install Dependencies
```powershell
cd mobile
npm install
```

### 2️⃣ Start Development Server
```powershell
npm start
```
Wait for the QR code to appear in the terminal.

### 3️⃣ Test on Your Phone (Easiest Method)

**Android Phone:**
1. Download "Expo Go" from Play Store
2. Open Expo Go → Scan QR Code
3. Scan the code from your terminal

**iPhone:**
1. Download "Expo Go" from App Store
2. Open Camera app (not Expo Go)
3. Point at QR code → Tap notification

**That's it!** The app will load on your phone.

## Troubleshooting

**Can't connect?**
- Ensure phone and PC are on same Wi-Fi
- Press `t` in terminal for tunnel mode

**Errors during install?**
```powershell
rm -rf node_modules
npm install
```

---

## Keyboard Shortcuts (in Expo terminal)

- `r` - Reload app
- `m` - Toggle menu
- `w` - Open in browser
- `a` - Open Android emulator (if installed)
- `t` - Toggle tunnel mode
- `j` - Open debugger
